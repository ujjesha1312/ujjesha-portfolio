# Portfolio Development Prompts

## üéØ Core Prompt for Portfolio Design
```
Create a modern, professional portfolio website for a Senior Full Stack Developer with:
- Dark theme with accent colors (blue/purple gradient)
- Smooth animations and micro-interactions
- Responsive design for all devices
- Sections: Hero, About, Skills, Projects, Experience, Contact
- Tech stack showcase with animated icons
- Project cards with hover effects
- Testimonials/recommendations
- Blog section (optional)
- Performance optimized (90+ Lighthouse score)
```

## üìù Section-Specific Prompts

### Hero Section Prompt
```
Design a hero section that:
- Has an engaging headline emphasizing "Senior Full Stack Developer"
- Includes a professional subtitle about specialization
- Features animated typing effect or gradient text
- Has CTA buttons (View Projects, Contact Me)
- Displays social links (GitHub, LinkedIn, Twitter)
- Includes a professional photo or avatar
- Has subtle background animations (particles, gradients, or geometric shapes)
- Shows scroll indicator
```

### About Section Prompt
```
Create an about section with:
- Professional bio (2-3 paragraphs)
- Years of experience
- Key achievements (numbers, metrics)
- Personal interests
- Download resume button
- Profile image with border effects
- Timeline of career highlights
```

### Skills Section Prompt
```
Build a skills section featuring:
- Categorized skills (Frontend, Backend, DevOps, Tools)
- Animated skill cards with icons
- Proficiency indicators
- Hover effects showing years of experience
- Tech logos with tooltips
- Skill tags with smooth animations
```

### Projects Section Prompt
```
Design a projects showcase with:
- Featured projects (3-6 cards)
- Project images/screenshots
- Tech stack badges
- Live demo and GitHub links
- Filter by technology
- Case study modal or detail page
- Metrics (users, performance improvements)
```

### Experience Section Prompt
```
Create an experience timeline with:
- Job positions in reverse chronological order
- Company logos
- Date ranges
- Key responsibilities (3-5 bullet points)
- Technologies used
- Achievements with metrics
- Interactive timeline visualization
```

### Contact Section Prompt
```
Build a contact section with:
- Contact form (name, email, message)
- Form validation
- Success/error states
- Email and phone display
- Social media links
- Location (optional)
- Availability status
- Alternative contact methods
```

## üé® Design System Prompts

### Color Scheme
```
Primary: Blue (#3B82F6)
Secondary: Purple (#8B5CF6)
Accent: Cyan (#06B6D4)
Background: Dark (#0A0A0A)
Surface: (#1A1A1A)
Text: White (#FFFFFF)
Muted: Gray (#9CA3AF)
```

### Typography
```
Headings: Geist Sans (Modern, Clean)
Body: Geist Sans
Code: Geist Mono
Sizes: 
  - Hero: 4xl-6xl
  - H1: 3xl-4xl
  - H2: 2xl-3xl
  - Body: base-lg
```

### Animation Guidelines
```
- Entrance animations: Fade in + Slide up (0.5s ease-out)
- Hover effects: Scale 1.05, add glow/shadow
- Page transitions: Smooth fade (0.3s)
- Loading states: Skeleton screens
- Scroll animations: Intersection Observer
```

## üöÄ Technical Implementation Prompts

### Performance Optimization
```
- Implement lazy loading for images
- Use Next.js Image component
- Code splitting by route
- Minimize bundle size
- Add loading states
- Optimize fonts with next/font
- Implement service worker for caching
```

### SEO Optimization
```
- Add proper meta tags
- Implement Open Graph tags
- Add JSON-LD schema
- Create sitemap.xml
- Add robots.txt
- Optimize images with alt text
- Implement proper heading hierarchy
```

### Accessibility
```
- ARIA labels for interactive elements
- Keyboard navigation
- Focus indicators
- Screen reader support
- Proper color contrast (WCAG AA)
- Skip to content link
```

## üí° Content Prompts

### Bio Writing
```
Write a compelling bio that:
- Starts with your unique value proposition
- Highlights 2-3 key achievements with numbers
- Mentions your specialization areas
- Shows personality and passion
- Includes a call-to-action
- Is 150-200 words
```

### Project Descriptions
```
For each project, include:
- One-line catchy description
- The problem it solves
- Your specific role
- Technologies used
- Key features (3-5)
- Measurable results (if available)
- Challenges overcome
```

### Meta Description
```
[Your Name] - Senior Full Stack Developer specializing in [tech stack]. Building scalable web applications with [X] years of experience. View my portfolio of projects in [specializations].
```
